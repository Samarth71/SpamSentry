# SpamSentry
 fraud detection platform utilizing pattern-matching algorithms to score and flag malicious messages/Emails 

SpamSentry is a web-based application designed to protect users from online fraud, phishing attempts, and deceptive messages. In today's digital world, scam emails and fraudulent text messages have become highly sophisticated, often tricking people into sharing sensitive information. This project provides a simple yet effective solution by allowing users to verify the legitimacy of any suspicious text before taking action.

The website works by taking the text of an email or message as input and analyzing it for known fraudulent indicators. It evaluates the content based on specific psychological triggers commonly used by scammers, such as artificial urgency, fake lottery wins, and unauthorized account suspension alerts. Along with analyzing text structures, the application also checks embedded links for deceptive or suspicious domain names.

After processing the text, SpamSentry generates a real-time risk score from 0% to 100%. This score helps users quickly understand how dangerous or safe the message is. Additionally, the system highlights the exact patterns and phrases that caused the warning, giving users a clear explanation of why the message was flagged. 

The goal of SpamSentry is to raise security awareness and offer a quick tool to double-check communication, making the internet a safer place for everyone.

## Features
* **Text Analysis:** Scans emails and messages for scam keywords.
* **Pattern Detection:** Identifies psychological manipulation and high-pressure language.
* **Risk Scoring:** Computes a clear safety percentage score.
* **User-Friendly Interface:** Simple text area for quick copy-paste scanning.
